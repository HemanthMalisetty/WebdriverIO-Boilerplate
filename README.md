# WebIO-Shop
WebdriverIO test framework


# Commands to run test cases

## For large web
npm run test -- -p [ PageName ] -b large

## For medium web
npm run test -- -p [ PageName ] -b medium

## For small web
npm run test -- -p [ PageName ] -b small

## For xsmall web
npm run test -- -p [ PageName ] -b xsmall
